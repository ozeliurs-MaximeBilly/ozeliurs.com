How I made a honeypot to catch Hackers.

02/08/2021

Come here to learn how to catch hackers, see what malware the execute currently and display this information beatifully using Flask.

As a holiday project I wanted to know why so many ip adresses tried to connect to my personnal server. In the quest of truth I stumbled on Cowrie, a linux program that captures ssh sessions and allows us to analyse the ssh traffic to our machine.

For installation you can install the app on docker which eases the install process. You can find info here. Otherwise you can install it normally on your machine with the help of this guide.
